# Pak RW v10.10.94 — KTP Design Studio Dashboard

Update ini hanya memperluas dashboard KTP Warga.

- Editor KTP satu halaman dengan alur Channel → Media → Tata Letak → Preview → Simpan → Test.
- Posisi judul, subjudul, data warga, foto, tanggal, status, dan footer dapat diatur manual.
- Ukuran font, warna, ukuran foto, radius, border, jarak baris, dan lebar teks dapat diedit.
- Upload background PNG/JPG/WebP maksimal 6 MB.
- Tambah banyak image dekorasi, atur posisi, ukuran, opacity, rotasi, visibilitas, dan hapus.
- Preview langsung 1011 × 639 di dashboard.
- Asset upload dibatasi ke folder assets/ktp-uploads dan divalidasi sebagai gambar.
- Renderer Discord membaca config desain yang sama dengan preview dashboard.
- Command rwktp tetap di semua channel dan rwktppanel tetap khusus channel KTP.
- Tidak mengubah AI, AFK Voice, level, role, poin, MongoDB, atau data KTP warga.
